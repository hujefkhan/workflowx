# Workflowx - Employee & Task Management System

Workflowx is a PHP & MySQL based employee and task management system where employees can request leave and managers/admins can assign and manage tasks efficiently.

---

## 🚀 Features

- Employee Authentication System
- Admin Dashboard
- Employee Dashboard
- Task Assignment System
- Leave Request Management
- Session Management
- MySQL Database Integration
- Simple & Responsive UI

---

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP

## 🚀 How to Run Locally

### 1️⃣ Clone Repository

```bash
git clone git@github.com:hujefkhan/workflowx.git
```

Move the project folder into:

```text
C:\xampp\htdocs\
```

---

### 2️⃣ Start XAMPP Services

Start:
- Apache
- MySQL

from XAMPP Control Panel.

---

### 3️⃣ Create Database

Open:

```text
http://localhost/phpmyadmin
```

Create a database named:

```text
workflowx_db
```

Import:

```text
database/workflowx_db.sql
```

---

### 4️⃣ Run Project

Open in browser:

```text
http://localhost/workflowx
```
